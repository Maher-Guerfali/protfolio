Project Overview
Video : https://youtu.be/vi2ERG7riZUI developed this website to showcase the Pingoos NFT Ordinals Collection, a collection created by my friend. Each unique Pingoos character is encoded as a Bitcoin Ordinal, making it a distinctive part of the NFT space.I built the site using Next.js, Tailwind CSS, and WebGL, ensuring fast performance, a clean design, and interactive 3D models of the Pingoos characters.In addition to the NFT collection, I integrated and published several WebGL games that I created using Unity. Visitors can play these games online and compete on an Azure-hosted leaderboard.
Check out the Pingoos NFTs and the games I developed, all available on this platform.

Project Overview
BrilliantXR is an innovative online platform that brings together cutting-edge WebXR features, offering everything from real-time video streaming and editing to immersive AR and VR experiences. Currently, only the landing page is available as the full platform is still under development, but exciting features are on the way.
‍Technologies Behind BrilliantXR:
Next.js for the frontend, providing a sleek and responsive user interface.
‍WebRTC for real-time video data transfer, ensuring smooth and fast video streaming experiences.
‍AWS Cloud Infrastructure to host the powerful backend, including:
‍TensorFlow for AI-powered video analysis and editing capabilities.
‍Unity running on the cloud to render complex AR/VR experiences.
‍Socket.IO and FastAPI are used to facilitate real-time communication between the server and clients, ensuring seamless performance across all interactions.Stay tuned for the full release of BrilliantXR, where you can explore the future of interactive and immersive media!

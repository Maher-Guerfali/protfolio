VR FPS Game for Quest2/3
Project Overview
I developed a dynamic VR first-person shooter game for Second Studio UK, designed to immerse players in an intense survival experience. In this game, you must face endless waves of enemies, with each wave becoming increasingly challenging.
The game features:
- Active Rigid Body Enemies: Enemies are equipped with realistic physics, simulating true-to-life body contact and reactions during combat.
- Special Weapons & Power-ups: Unlock powerful weapons and tactical abilities to enhance your survival.
- Unlockable Zones: Explore and unlock new areas as you progress through the waves.
- Survival Mechanics: Fight against hordes of enemies in various stages, testing your endurance and strategy.
This game pushes the limits of immersion and physicality in VR. Contact me if you'd like to try it out and experience the action firsthand!
Tools and Plugins
- PuppetMaster: Enemy physics
- Polygone Arcade: Map assets
- Interaction Framework VR: XR hands

First Aid Simulation
Quest2/Quest3
B2B project
Project Overview
I developed a VR first aid simulation for Tadreex, a Tunisia-based VR application startup, as part of my internship. This serious game features 5 real-life scenarios to teach critical first aid skills, including lateral positioning, using a defibrillator, fire extinguisher operation, and handling toxic materials. The game provides an immersive, hands-on learning experience for practical first aid training.
Tools and Plugins
Using React, I turned a Figma design into a fully functional site, built with Next.js and Tailwind CSS.

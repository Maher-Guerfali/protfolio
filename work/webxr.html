Endless Runner Pingoos
IOS/Android/Webgl
Project Overview
Pingoos Endless Runner is a mobile app available for iOS and Android, with a separate optimized WebGL version for playing in the browser. It's an endless runner game packed with exciting features, including various obstacles, spell powers, frost VFX and sound effects, coin magnets, shields, and much more.I fully developed this game from the ground up, handling everything from scripting and game mechanics to designing, 3D modeling, and rigging. The game delivers a seamless experience across both mobile and web platforms, offering players a unique and dynamic take on the endless runner genre.
My Contributions
I created Pingoos Endless Runner entirely, handling every aspect of development. This includes UI/UX design, 3D modeling and rigging, sound effects, animation,object pooling and implementing the linking system.
Endless Runner Game - Key Features:
Fluid & Smooth Gameplay: Optimized for seamless performance, ensuring players experience fast-paced action without any lag or interruptions.
‍Efficient Pool System: All objects, including obstacles and collectibles, use a pooling system, eliminating the need for runtime instantiation and enhancing overall performance.
‍Diverse Obstacles: Featuring over 20 unique obstacles, each with custom collision mechanics and behavior, providing an ever-changing challenge with each run.
‍Custom Skin System: Players can choose from a variety of character skins, which load dynamically during gameplay, offering personalization without affecting performance.

Youbama Online
Online Webgl
Project Overview
Key Features:
Authentication System: Secure login and registration built using Node.js and MongoDB, with a password reset feature handled by Nodemailer for seamless user management.
‍2D Multiplayer Gameplay: Youbama offers two exciting modes:
‍Capture the Flag and Basketball Game. In Capture the Flag, players must capture 3 flags to win, while the basketball mode challenges players to outscore their opponent in a fast-paced 1v1 match.
‍Real-Time Online Service: Multiplayer functionality powered by Photon PUN for smooth scene synchronization, ensuring players enjoy a lag-free experience in real-time.
‍Player Stats & Leaderboard: The main menu displays essential player information, including total games played, games won, and games lost. A global leaderboard showcases top players, their win/loss stats, and performance, allowing players to compete for the top ranks.
